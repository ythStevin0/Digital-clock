# Digital Clock - Python Tkinter

Simple digital clock application built using Python and Tkinter.

## Features

- Real-time clock display
- Updates every second
- Simple GUI interface
- Dark theme design

## Installation

1. Clone repository
```
bash
git clone https://github.com/ythStevin0/Digital-clock.git
```

2. Run the application
```
bash
python main.py
```

## Requirements

- Python 3.x
- Tkinter (included with Python)
